# 실거래 금지 정책

본 프로젝트는 투자 리서치 자동화 시스템이다.

## 금지 사항

- 실계좌 주문 실행
- 증권사 API 주문 연동
- 자동매매 실행
- 사용자 확인 없는 매수·매도 신호 발송
- 수익 보장 표현
- 손실 가능성 은폐
- 미확인 정보를 근거로 한 강한 매수 권고

## 허용 사항

- 종목 리서치 보고서 생성
- 재무 분석
- 뉴스 감성 분석
- 업종 분석
- 시나리오 분석
- 리스크 감사
- 관찰 종목 정리
- 투자 체크리스트 생성

## 향후 실거래 기능 확장 조건

실거래 기능은 다음 조건 충족 전까지 금지한다.

1. 별도 사용자 명시 승인
2. 법무·규정 검토
3. API 키 보안 설계
4. 주문 전 최종 확인 절차
5. 로그·복기·오류 대응 체계
6. 최대 손실 제한
7. 킬스위치
8. 모의투자 검증
