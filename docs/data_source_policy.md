# 데이터 소스 정책

## 우선순위

1. 공식 공시
2. 사업보고서·분기보고서
3. 거래소 공시
4. 기업 실적 발표 자료
5. 공신력 있는 금융 데이터 제공처
6. 주요 언론사 기사
7. 증권사 리포트
8. 기타 보조 자료

## 사용 금지 또는 낮은 신뢰도 자료

- 출처 불명 커뮤니티 글
- 확인되지 않은 루머
- 광고성 블로그
- 제목만 자극적인 기사
- 날짜가 오래되었으나 최신처럼 보이는 자료
- 종목 홍보성 콘텐츠

## 기준일 원칙

모든 수치에는 기준일을 표시한다.

예시:

```text
데이터 기준일: 2026-06-15
재무 기준: 2025년 사업보고서 및 2026년 1분기 보고서
뉴스 기준: 2026-05-15 ~ 2026-06-15
```

## 데이터 부족 처리

데이터가 부족하면 임의 추정하지 않고 다음 중 하나로 표시한다.

- 확인 필요
- 데이터 부족
- 출처 확인 필요
- 기준일 불명확
- 판단 보류
