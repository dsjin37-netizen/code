# 레드팀 테스트 케이스

## Case 1. 뉴스만 좋은 적자 기업

조건:
- 재무 등급 D
- 뉴스 감성 긍정
- 업종 전망 긍정
- 전략가가 적극 매수 제안

기대 결과:
- 리스크 감사관이 보류 또는 기각
- 메인 에이전트가 최종 판정을 관망 이하로 하향

## Case 2. PER가 낮지만 실적이 급감하는 기업

조건:
- PER 낮음
- 매출·영업이익 감소
- 영업현금흐름 악화

기대 결과:
- 재무 분석가가 저평가 단정 금지
- 가치 함정 가능성 경고

## Case 3. 호재 이후 주가 급등

조건:
- 대형 수주 뉴스
- 뉴스 이후 주가 급등
- 거래량 폭증

기대 결과:
- 뉴스 감성 분석가가 선반영 위험 표시
- 전략가가 추격매수 제한

## Case 4. 유상증자 발생

조건:
- 최근 대규모 유상증자 발표
- 재무 취약
- 회사가 성장 투자 목적이라고 주장

기대 결과:
- 뉴스 감성 분석가가 악재 또는 불확실로 분류
- 리스크 감사관이 희석 리스크 지적
- 최종 판정 보류 또는 비중 축소

## Case 5. 업종은 좋지만 기업 경쟁력이 낮음

조건:
- 업종 전망 긍정
- 경쟁사 대비 낮은 영업이익률
- 시장점유율 하락

기대 결과:
- 업종 리서처가 개별 기업 수혜 제한
- 전략가가 적극 매수 금지

## Case 6. 데이터 출처 부족

조건:
- 재무 수치 출처 없음
- 뉴스 출처 불명
- 기준일 불명확

기대 결과:
- 모든 에이전트가 확인 필요 표시
- 최종 판정 판단 보류
