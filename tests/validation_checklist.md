# 검증 체크리스트

## 입력 검증

- [ ] 종목명 또는 티커가 있는가?
- [ ] 시장 구분이 있는가?
- [ ] 투자기간이 있는가?
- [ ] 보유 여부가 있는가?
- [ ] 기준 가격이 있는가?
- [ ] 데이터 기준일이 있는가?

## 재무 분석 검증

- [ ] 최근 3개년 실적이 있는가?
- [ ] 최근 4개 분기 실적이 있는가?
- [ ] 매출·영업이익·순이익이 분리되어 있는가?
- [ ] 영업현금흐름을 확인했는가?
- [ ] 재고자산·매출채권 이상 증가를 확인했는가?
- [ ] 부채비율과 유동비율을 확인했는가?
- [ ] 밸류에이션을 동종업계와 비교했는가?

## 뉴스 분석 검증

- [ ] 최근 1개월 뉴스 기준인가?
- [ ] 공식 공시와 언론 보도를 구분했는가?
- [ ] 호재·악재·중립·불확실을 분류했는가?
- [ ] 선반영 위험을 확인했는가?
- [ ] 치명적 리스크 이벤트를 확인했는가?

## 업종 분석 검증

- [ ] 정확한 사업부문을 확인했는가?
- [ ] 글로벌 시장 흐름을 확인했는가?
- [ ] 경쟁사를 비교했는가?
- [ ] 정책·규제 변화를 확인했는가?
- [ ] 업종 전망과 개별 기업 수혜를 분리했는가?

## 전략 검증

- [ ] 최종 판정 5단계 중 하나로 제시했는가?
- [ ] 목표 수익률이 있는가?
- [ ] 손절 라인이 있는가?
- [ ] 포지션 크기가 있는가?
- [ ] 진입·추가매수·청산 조건이 있는가?
- [ ] Bull/Base/Bear 시나리오가 있는가?

## 리스크 감사 검증

- [ ] 매수 논리의 약점을 공격했는가?
- [ ] 누락 리스크를 제시했는가?
- [ ] 최악 시나리오를 제시했는가?
- [ ] 감사 판정이 있는가?
- [ ] 감사 결과가 최종 판정에 반영되었는가?

## 최종 보고서 검증

- [ ] 핵심 결론이 명확한가?
- [ ] 현재 판정이 명확한가?
- [ ] 반례와 무효화 조건이 있는가?
- [ ] 투자 유의 문구가 있는가?
- [ ] 출처 없는 단정 표현이 없는가?
